#ko-ch1
Demo Typescript Knokoutjs Client App

####Uses
   - Tsc Transpiler
   - amd loader
   - bower for Client Dependency management
```
        bower i
```
   - npm for Typescript typedef Dependency management
```   
        npm i
````
#### To Compile and Run
   - install npm module typescript globally
   - install npm module http-server globally
```
    tsc
    http-server
```
